# NodeJS Authentication System
This is a user login and registration application using Node.js.
## Usage 
>$ npm install 

>$ npm start  or npm run dev

>$ visit http://localhost:5000

## Libraries & tools used:
* Bootswatch v4.3.1
* MongoDb Atlas v4.0.12
* Bootstrap
* FontAwesome
* Express
* Mongoose 5.7.3
* Passport
